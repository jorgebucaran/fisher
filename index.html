--- layout: default title: Fisherman ---

The plugin manager and CLI toolkit for fish

An Open Source project by the Ocean. Made with by Fishermen

Easy

Share and reuse code, prompts and configurations easily.

Social

Compatible with plugins and prompts from others like Tackle, Oh My Fish! and Wahoo.

Tools

Create powerful CLI utilities with Fishtape, Getopts and Wait.

What makes Fisherman great?

Flat-Tree Model

External self-managed index

Unified plugin architecture

Dependency manifest file

Built-in cache system

Get started with Fisherman

Prepare to be boarded:

# Clone the booty
$ git clone https://github.com/fisherman/fisherman

# Board the repository
$ cd fisherman

# Install Fisherman
$ make

Also checkout the ASCII screencasts.
